# Nishimacros

## WELCOME

This repository contains FIJI and FIJI-associated macros and scripts used for reproducible image processing in FIJI. Typical input is a directory of .dv or .tif files. Typical output is either 1) processed image files, or 2) tabular data

## TABLE OF CONTENTS

[01_makeContactSheet](https://github.com/erinosb/Nishimacros/tree/main/01_makeContactSheet)

Starting with a directory of .dv files, create an arranged contact sheet of z-projected images.

[02_quantifyImageRegion](https://github.com/erinosb/Nishimacros)

Starting with a directory of .dv files, measure the total internal fluorescence intensity of different channels along a long rectangle. Report the quantifications as y= intensities over x = length of the rectangle as tabular data.
